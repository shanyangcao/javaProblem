## Spring Boot 核心依赖

### Spring Boot Starter

```
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter</artifactId>
</dependency>
```

**含义**：Spring Boot核心启动器，包含自动配置、日志、YAML支持等。

### Spring Boot Web Starter

```
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-web</artifactId>
</dependency>
```

**含义**：用于构建Web应用，包含Spring MVC、Tomcat服务器、RESTful支持。

## 数据访问依赖

### Spring Data JPA 

```
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-data-jpa</artifactId>
</dependency>
```

**含义**：简化数据库操作，提供ORM（对象关系映射）功能。

### MySQL 驱动

```
<dependency>
  <groupId>mysql</groupId>
  <artifactId>mysql-connector-java</artifactId>
</dependency>
```

**含义**：连接MySQL数据库的JDBC驱动程序。

### MyBatis

```
<dependency>
  <groupId>org.mybatis.spring.boot</groupId>
  <artifactId>mybatis-spring-boot-starter</artifactId>
</dependency>
```

**含义**：持久层框架，通过XML或注解配置SQL映射。

## 消息队列依赖

### RabbitMQ

```
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-amqp</artifactId>
</dependency>
```

**含义**：集成RabbitMQ消息队列，用于异步处理、应用解耦。

## 缓存依赖

### Redis

```
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-data-redis</artifactId>
</dependency>
```

**含义**：集成Redis缓存，提高数据访问性能。

## 安全依赖

### Spring Security

```
<dependency>
  <groupId>org.springframework.boot</groupId>
  <artifactId>spring-boot-starter-security</artifactId>
</dependency>
```

**含义**：提供认证和授权功能，保护应用安全。

## 工具类依赖

### Lombok

```
<dependency>
  <groupId>org.projectlombok</groupId>
  <artifactId>lombok</artifactId>
  <optional>true</optional>
</dependency>
```

**含义**：通过注解自动生成getter、setter、构造函数等，减少样板代码。

### Apache Commons

```
<dependency>
  <groupId>org.apache.commons</groupId>
  <artifactId>commons-lang3</artifactId>
</dependency>
```

**含义**：提供各种工具类，如字符串处理、日期处理等。
